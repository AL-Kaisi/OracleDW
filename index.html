System Overview

The Green Care Provider Temps Data Warehouse is a comprehensive system for managing temporary care workers. Originally built on Oracle, it has been modernised with PostgreSQL and a Flask web interface.

Migration Benefits

  • Cost Reduction: PostgreSQL is free and open-source
  • Modern Interface: Web-based UI accessible from any device
  • Better Performance: Optimised queries and indexing
  • Easy Deployment: Docker containers for consistent environments
  • Enhanced Analytics: Interactive dashboards and reports
  • Scalability: Horizontal scaling with read replicas

Getting Started

  1. Clone the repository
  2. Run docker-compose up -d
  3. Access the application at http://localhost:5000
  4. Load CSV data using the Data Management interface
  5. Run the ETL process to populate the warehouse
